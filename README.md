## Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- An API key (OpenAI or Google Gemini)

### 1. Backend Setup

```bash
cd backend
cp .env.example .env
# Edit .env and add your API key (OPENAI_API_KEY or GOOGLE_API_KEY)

# Create virtual environment & install dependencies
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

# Start the backend server
uvicorn app.main:app --reload
```

The API will be available at **http://localhost:8000** with interactive docs at **http://localhost:8000/docs**.

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The UI will be available at **http://localhost:5173**.

### 3. Docker (Alternative)

```bash
docker compose up --build
```

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/` | API overview |
| GET | `/health` | Health check |
| POST | `/api/analyze` | Analyze symptoms (detailed) |
| POST | `/api/chat` | Chat-style symptom analysis |
| GET | `/api/history/{session_id}` | Get chat history |

### Example: POST /api/analyze

```json
{
  "description": "I have a headache and mild fever for 2 days",
  "age": 30,
  "gender": "male",
  "duration": "2 days",
  "severity": "mild"
}
```

## AI Providers

The app supports **OpenAI** (default) and **Google Gemini**:

```env
# Option 1: OpenAI
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini

# Option 2: Google Gemini
GOOGLE_API_KEY=...
GEMINI_MODEL=gemini-2.0-flash
LLM_PROVIDER=gemini
```

Without any API key, the app falls back to a basic rule-based analysis.
